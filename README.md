# ProEval AI Frontend

Advanced AI-powered student project evaluation and progress tracking system.

## 🚀 Project Overview

ProEval AI is a platform designed to streamline the academic project lifecycle, from enrollment and guide selection to phase-wise submissions and feedback tracking.

## 📁 Project Structure

The project follows a clean, scalable directory structure based on Next.js 15+ best practices:

```text
src/
├── app/                  # Next.js App Router (Routing & Pages)
│   ├── (auth)/           # Authentication routes (Login, Register)
│   ├── (portal)/         # Main application portal
│   │   └── student/      # Student-specific dashboards and tools
│   ├── layout.tsx        # Global root layout
│   └── page.tsx          # Marketing/Landing page
├── assets/               # Static assets (images, icons, etc.)
├── components/           # React Components
│   ├── common/           # Shared/Generic components (Layout, Navbar, etc.)
│   ├── features/         # Feature-specific complex components
│   └── ui/               # Reusable atomic UI components (shadcn)
├── constants/            # Application-wide constants and enums
├── hooks/                # Custom React hooks
├── lib/                  # External library configs and core utilities
│   ├── validations.ts    # Zod schemas for data validation
│   └── utils.ts          # Shared utility functions
├── services/             # API and external service logic
└── types/                # Global TypeScript type definitions
```

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI / Shadcn
- **State/Forms:** React Hook Form + Zod
- **Animations:** Framer Motion
- **Icons:** Lucide React / Tabler Icons

## 🏁 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```
